2. Tools and Plugins
Build on top of Twitter Bootstrap

The W Balls HTML template is build on top of SASS Twitter Boostrap HTML framework. All the components of the Bootstrap framework were left as they are during the development, so the template is upgrade-safe. The only file which has been changed and is a part of Bootstrap is _variables.scss, inside the
./lib/_variables.scss directory.

Find the documentation for Twitter Boostrap here.

Notes
 

You are allowed to use any component in your themes available in Twitter Bootstrap. However, for the performance reasons some of the components were commented out in the .lib/bootstrap.scss file. Uncomment the components you need in your project and recompile the project using $ compass compile when you cd to the root of the template in terminal.

Green (default)

Blue

Orange

You can find the home pages in different color variants in the files index.html, index-blue.html, index-orange.html and index-red.html.

How to Change the Theme

If you are familir with Compass/SASS, the easiest way to change the color theme is to change the variable $clr-theme in the beginning of the main.scss file, which is located in ./lib/main.scss . Just set the variable to the any valid CSS color. You can help yourself with a tool like HTML Color Picker. 

$theme-clr : #bada55;

If you are unfamiliar with the Compass/SASS authoring framework, you can change the color theme by calling the other CSS file. The default theme is "green". Just change the link to the other CSS file in head section of the HTML document from this: 

<link rel="stylesheet" href="stylesheets/main.css" />

to this: 

<link rel="stylesheet" href="stylesheets/orange.css" /><!--  or "blue.css" or "red.css"  -->

Boxed and Wide version

W Balls HTML has two different layout modes: boxed and wide.

Wide

Boxed

It is very easy to switch between the two. The default version is wide version. If you prefer to have a boxed version of the theme, the only thing you need to do is to add the class boxed to the main <body> tag.

Here is the example:

<body class="boxed">

Sticky or Static Navigation

W Balls has two types of the navigation: static or sticky.

The difference is that the sticky navigation will always stay at the top of the browser, no matter how deep we scroll down into the page. This way the user always have access to te main navigation menu.

On the other hand, the static navigation will stay on its position at the top of the page.

Default navigation style is sticky. However, if you want to switch to the static navigation, you have to do two things:

  1. Add the class no-sticky to the <body> tag:

    <body class="no-sticky">

  2. In the navigation div change the class navbar-fixed-top to the navbar-static-top (around line 40):

    <div class="navbar navbar-inverse navbar-static-top">

Carousels

You can turn your static images / HTML block to carousel using by writing the following code: 

<nav class="arrows pull-right">
      <a href="#" class="nav-left icon icons-arrow-left" id="sliderNavigationLeft"></a>
      <a href="#" class="nav-right icon icons-arrow-right" id="sliderNavigationRight"></a>
</nav>
<div class="carouFredSel" data-nav="sliderNavigation">
	<div class="slide">
		one item
	</div>
	<div class="slide">
		another carousel item
	</div>
</div>

Note the data-nav attribute in the div with a class "carouFredSel". You have to put here the ID of the navigation arrows, without the 'Right' and 'Left' at the end, as shown above.

Flickr Photo Stream

In the blog.html file you will notice the Flickr Photo Stream. This can be changed to your own stream. In the file js/custom.js find the comment // = Flickr Feed = (around line 500, at the end of the file) and change the configuration of the plugin.

 $('#flickrGallery').jflickrfeed({
        limit: 6, // number of images shown
        qstrings: {
            id: '52617155@N08' // ID of the Flickr user
        },
        // ...

More info about the configuration settings for this plugin you can find HERE.

PrettyPhoto Lightbox Clone

Many images on the website open in the lightbox clone named PrettyPhoto. The documentations for the PrettyPhoto can be found here. You can add the PrettyPhoto to almost any web element (image, video, swf, iframe etc.).

The easiest way with the default configuration is to add the .add-prettyphoto class to the link tag:

<a href="url/to/pic.jpg" class="add-prettyphoto">
   <img src="url/to/pic.jpg" alt="image">
</a>

For more advanced use you can create your triggers with custom settings. The details can be found on the link above to the documentation of the PrettyPhoto.

Google Map

In the contact.html/contact.php page you can set the location of your business to be displayed on the Google Maps.

W Balls HTML template use the $.goMap() jQuery plugin for calling the Google Maps API and displaying the map with a custom pin.

To change the location, open the file js/custom.js and find these lines:

$('#gmap').goMap({
    markers : [{
        address : 'Leatherhead, Surrey United Kingdom 01372 818123, United Kingdom', /* change your

 adress here */
        title : 'W Balls HTML Theme - ThemeForest', /* title for the marker */
        icon : {
            image : 'images/pin_red.png' /* your custom icon file */
        }
    }],
    scrollwheel : false,
    zoom : 13,
    maptype : 'ROADMAP'
});

6. Contact Form

W Balls HTML comes with a contact.php file with actually working form, so your clients can contact you directly to your email or request a quote for their project.

The form is written in PHP and can be found in ./contact.php . Be sure that your server supports PHP and has all the required settings for build-in mail() PHP function.

All you need to do to enable the PHP form is to point the user to the contact.php instead of contact.html and change the following variable in the contact.php to your actual email address: 

/*...*/
$to = "your@email.com"; /* you email address */

The message will be sent directly from your website to the email address you have specified above and will have the following form:
